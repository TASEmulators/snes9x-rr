Miscellaneous


Where the Files are Stored

You can choose where Snes9x will look for cheats (.cht), SRAM files (.srm), freeze files (.frz), screenshots (.png), etc in 'File' tab of 'Preferences' dialog.

  • Snes9x Folder the subdirectories will be searched in the same folder as Snes9x.
    ROM Folder the files will all be in the same folder as the ROM being used.
    Application Support Folder the subdirectories will be created in /Users/yourname/Library/Application Support/Snes9x and searched there.

Which Display is Used in Full Screen Mode

If you use two or more displays, in full screen mode Snes9x uses the display where the game window is shown.


Using Graphics Packs

The games which have SPC7110 chip in their cartridges need a set of files called Graphics Pack. To use Graphics Packs,

  1. If you select 'Snes9x Folder' or 'Application Support Folder' in 'Preferences' dialog, create a 'Packs' folder in that directory.
  2. Put the Graphics Pack folder (or its alias) into the 'Packs' folder. if you select the 'ROM Folder' option in 'Preferences' dialog, put it into the same directory as the ROM image.
  3. Rename the name of the Graphics Pack folder (or its alias) to the one listed below. Or else you can use the same name as the ROM image (except extension), for example if your ROM image is 'FEOEZ.smc', Graphics Pack folder can be named 'FEOEZ'.
  • Super Power League 4 (J) SPL4-SP7
    Momotarou Dentetsu Happy (J) SMHT-SP7
    Far East Eden of Zero (J) FEOEZSP7
    Far East Eden of Zero Shonen Jump no Shou (J) SJUMPSP7

Note that users who use UFS disk format should take care of upper/lower case. The file name itself is upper case, but its extension is lower case.

The games which have S-DD1 chip in their cartridges no longer need Graphics Packs, but you can still use them for speed improvement.

  • Star Ocean (J) SOCNSDD1
    Street Fighter Alpha 2 (US) SFA2SDD1
    Street Fighter Alpha 2 (E) SFA2SDD1.E
    Street Fighter Zero 2 (J) SFZ2SDD1

Using IPS Patch

Snes9x automatically patches without overwriting the ROM image.

  1. If you select 'Snes9x Folder' or 'Application Support Folder' in 'Preferences' dialog, create 'IPSs' folder in that directory.
  2. Put the IPS patch file into the 'IPSs' folder. if you select 'ROM folder' in 'Preferences' dialog, put it into the same directory as the ROM image.
  3. Rename the name to the same as the ROM image (except extension, it is '.ips').
  4. If you want to use multiple IPS files at a time, set their extensions to '.000.ips', '.001.ips', ...

SPC File Dumping

SPC file (.spc) is the freeze state of only sound part, so it is known as SNES music data file. Dump the SPC file by Snes9x and play the file by SPC players like Audio Overload and PSFPlay.
Assign your gamepad button or keyboard to 'SPC' button in 'Configure Controllers' or 'Configure Keyboard' dialog. To dump the SPC file, press 'SPC' button while playing a game, or choose 'Save SPC File at Next Note-on' in 'Option' menu and resume the game.


Music Box

Music Box is to play the game music without playing the game. Pause the game and enter Music Box by 'Music Box...' in 'Option' menu, and you can enjoy the music with doing other things like surfing Internet.
There are two modes for playing music. Select in 'Others' tab in 'Preferences' dialog.

  • Sound Emulation Only in a shooting game, you may listen BGM endlessly. (Music plays.)
    Whole Emulation in a shooting game, you may hear explosion sound and gameover music. (Music plays and the CPU continues to execute the emulation.)

Each mode stores the emulation state before entering Music Box, and restores after closing Music Box. 'Whole Emulation' makes a temporary freeze file.


Sound Effect

You can use Apple's Matrix Reverb / Graphic Equilizer and enhance the game sounds. Call 'Sound Effect' dialog from 'Preferences' / 'Music Box' dialog.


Core Image Filters

In Mac OS X 10.4 or later, some of the Core Image filters can be applied to the game screen. Turn on 'Use Core Image' option in 'Preferences' dialog and choose 'Core Image Filter...' menu while playing a game.
